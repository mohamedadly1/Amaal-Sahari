# Admin Panel Guide

## Overview

The Amaal Sahari Admin Panel is a comprehensive content management system (CMS) that allows you to control and manage all content across the website without touching code.

## Access

- **URL**: `/admin`
- **Username**: `admin`
- **Password**: `amaal2024`

## Admin Panel Structure

### Header
The sticky header at the top of the admin panel contains:
- **Logo & Title**: Quick identification of the admin panel
- **Reset Button**: Reset all content to default values (use with caution)
- **View Site Button**: Preview the public website in a new tab
- **Logout Button**: Exit the admin panel

## Content Management Sections

### 1. **Navbar** (Menu)
Control your website's main navigation bar.

**What you can manage:**
- Logo image and sizing
- Navigation menu items (English & Arabic)
- Color scheme (background, text, hover, accent colors)
- Call-to-action button text

**Best Practice**: Ensure colors have sufficient contrast for accessibility.

## Common Tasks

### Adding New Content
1. Navigate to the relevant section
2. Click the "+" or "Add" button
3. Fill in the required fields
4. Click "Save Changes"

### Editing Existing Content
1. Find the content item in the list
2. Click to expand or edit the field
3. Make your changes
4. Click "Save Changes"

### Deleting Content
1. Locate the item you want to delete
2. Click the trash/delete icon
3. Confirm the deletion
4. Click "Save Changes"

### Bilingual Content
Most sections support both English and Arabic:
- Fill in the English version first
- Use the Arabic language toggle or separate Arabic field
- Ensure both versions are translated before saving

## Best Practices

### Content Quality
- **Be Consistent**: Use consistent terminology and tone
- **Be Specific**: Include concrete numbers and details
- **Be Clear**: Use simple language that's easy to understand
- **Be Current**: Keep information up-to-date

### Images
- Use high-quality images (minimum 1200px width)
- Optimize file sizes before uploading
- Ensure images are relevant and professional
- Include descriptive alt text

### SEO
- Use keywords naturally in descriptions
- Keep meta descriptions under 160 characters
- Use unique titles for each page
- Add relevant tags and categories

### Bilingual Content
- Translate accurately, not just word-for-word
- Maintain tone and style in both languages
- Test both language versions on the website
- Consider cultural nuances in translations

### Before Publishing
1. Preview changes using "View Site" button
2. Check both language versions (EN & AR)
3. Verify all links work correctly
4. Test on mobile devices
5. Ensure images load properly
6. Review for spelling and grammar

## Troubleshooting

### Changes Not Saving
- Ensure all required fields are filled
- Check for error messages
- Try saving again
- Clear browser cache and refresh

### Content Not Appearing on Website
- Verify you clicked "Save Changes"
- Ensure the content is not hidden/archived
- Check if the content is set to display
- Review preview to confirm changes

### Language Version Issues
- Ensure both EN and AR versions are filled
- Verify correct language selection
- Check RTL (Right-to-Left) text direction for Arabic
- Test on the live website

## üÜò Troubleshooting

### Problem: Changes not showing on website
**Solution:**
1. Make sure you clicked "Save Changes"
2. Wait for "Saved!" message
3. Refresh the website (Ctrl+R or Cmd+R)
4. Clear browser cache if still not showing

### Problem: Arabic text appears backwards
**Solution:**
1. Ensure you're in the Arabic (AR) field
2. Check that field has `dir="rtl"` attribute
3. Some browsers need refresh to display correctly
4. Test in different browser

### Problem: Image not showing
**Solution:**
1. Verify the image URL is correct
2. Copy URL from web browser address bar
3. Check image format (JPG, PNG, WebP)
4. Try a different image to test
5. Ensure image file is publicly accessible

### Problem: Can't save changes
**Solution:**
1. Check all required fields are filled (red asterisk *)
2. Look for error messages in red
3. Try saving again
4. Clear browser cookies/cache
5. Try a different browser
6. Contact your developer if still stuck
